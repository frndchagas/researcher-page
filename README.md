## Researcher page example using Tailwindcss

## About

This example includes:

- Cover picture and detais about the researcher.
- Researcher's publications
- Researcher's disciplines and research areas
- Researcher's contact information
- Researcher photos
- Contact
- Social media

## Installation

A step by step list of commands / guide that informs how to install an instance of this project.

- Clone this repository

`$ git clone https://github.com/frndchagas/researcher-page.git`

- Install npm dependencies

`$ npm install`
